Sobre mí

Me encanta combinar mi creatividad con mis conocimientos tecnológicos para ofrecer soluciones web a medida para cada cliente. Tengo más de 3 años de experiencia trabajando con diferentes plataformas, metodologías y herramientas como lo son Laravel, DJango, MCV(Modelo vista controlador), 
adicionalmente, algunas con las que he trabajado son:

JavaScript

JavaScript crea contenido web dinámico e interactivo en el navegador. Es muy popular y versátil. 

API

API conecta aplicaciones de software con protocolos y herramientas. API usa funcionalidades o servicios sin ver el código. API mejora el desarrollo de software moderno. 

MERN

MERN usa MongoDB, Express, React y Node para crear aplicaciones web. Estas tecnologías crean sitios web flexibles y modernos. MERN sigue el patrón MVC, que divide la aplicación en tres partes. 

Angular

Angular crea aplicaciones web con JavaScript, TypeScript y MVC. Estas aplicaciones son flexibles, modernas y fáciles de mantener.
